## 🚀 Como executar

- Clone o repositório
- Instale as dependências com `yarn` ou `npm install`
- Inicie o servidor com `yarn start` ou `npm start`

Agora você pode acessar [`localhost:3000`](http://localhost:3000) do seu navegador.
